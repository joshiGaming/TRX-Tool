# trx-merge

Merge aus mehreren TRX Dateien (Joshua Honz / Praktikant)

[XmlMerge in Nodejs](https://ulm-dev.zuken.com/jhonz/XmlMergerNodejs)

## arguments

1:
`./XmlMerge.exe OutputDirectory InputDirectory`

2:
`./XmlMerge.exe -help`
